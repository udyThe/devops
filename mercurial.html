Mercurial

A guide to using Mercurial for distributed version control.

Installation

To install Mercurial, run the following command:

sudo apt-get install mercurial

Basic Mercurial Commands

Initialize a Repository

hg init

Add Files

hg add file.txt

Commit Changes

hg commit -m "Initial commit"

Push Changes

hg push

Pull Changes

hg pull

Branching and Merging

Create a new branch:

hg branch new-feature

Merge branches:

hg merge new-feature
