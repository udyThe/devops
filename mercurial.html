Mercurial

A guide to using Mercurial for distributed version control.

What is Mercurial?

Mercurial is a distributed version control system (DVCS) designed for efficient handling of projects of any size. It is fast, lightweight, and easy to use.

Installation

To install Mercurial on your system, use the following command: 

sudo apt-get install mercurial

Basic Mercurial Commands

Initialize a Repository

To create a new Mercurial repository, use the hg init command: 

hg init

Add Files

To start tracking a file in the repository, use the hg add command: 

hg add file.txt

Commit Changes

To save changes to the repository, use the hg commit command with a commit message: 

hg commit -m "Initial commit"

Branching and Merging

Mercurial allows you to create branches to work on new features or experiments. To create a new branch, use: 

hg branch new-feature

To merge changes from one branch into another, use the hg merge command: 

hg merge new-feature
